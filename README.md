# 마케팅 KPI AI Insight 분석

GA4 BigQuery 데이터를 기반으로 이벤트별 마케팅 KPI를 자동 집계하고,  
Vertex AI Gemini로 실행 액션 중심의 인사이트를 생성하는 마케팅 성과 분석 대시보드입니다.

이 프로젝트는 단순 리포트가 아니라,  
**“어떤 이벤트가 성과를 만들고 있는지, 어떤 이벤트를 개선해야 하는지, 다음 액션은 무엇인지”**를 빠르게 판단하기 위해 구축했습니다.

> GA4 데이터 수집 → BigQuery KPI 집계 → 유저 군집 분석 → Vertex AI 인사이트 생성 → Looker Studio 시각화까지 자동화한 프로젝트입니다.

## Problem

마케팅 이벤트 페이지는 유입, 참여, 구매, 매출 성과가 각각 다르게 나타납니다.  
하지만 기존 방식으로는 이벤트별 성과를 수동으로 확인하고 해석해야 했기 때문에 다음과 같은 문제가 있었습니다.

- 이벤트 유입은 많지만 실제 구매로 이어지는지 즉시 판단하기 어려움
- 3개월, 1개월, 주차별 성과 흐름을 한눈에 비교하기 어려움
- 이벤트가 유입용인지, 구매 전환용인지, 참여 유도용인지 구분하기 어려움
- 데이터는 많지만 실제 마케팅 액션으로 연결되는 해석이 부족함
- 담당자가 매번 수동으로 인사이트를 작성해야 함

---

## Solution

이 프로젝트는 GA4, BigQuery, Vertex AI, Looker Studio를 연결하여  
이벤트별 마케팅 성과 분석과 인사이트 생성을 자동화했습니다.

핵심 해결 방식은 다음과 같습니다.

- GA4 이벤트 데이터를 BigQuery에서 이벤트 단위 KPI로 집계
- 기간별 성과를 3개월, 1개월, 주차 단위로 비교
- 페이지 반응과 전환율을 기준으로 이벤트를 사분면에 배치
- 유저 군집 분석을 통해 이벤트 방문자의 성향 반영
- Vertex AI Gemini를 활용해 이벤트별 자연어 인사이트 생성
- Looker Studio에서 실무자가 바로 확인할 수 있는 대시보드 구성

---

## Key Features

### 1. 이벤트별 KPI 자동 집계

이벤트별로 다음 지표를 자동 집계합니다.

| 지표 | 설명 |
|---|---|
| Users | 이벤트 방문 유저 수 |
| UX Score | 페이지 반응 점수 |
| Purchase CVR | 구매 전환율 |
| Interaction CVR | 참여 전환율 |
| Revenue | 매출 |
| ARPPU | 구매자당 평균 매출 |
| Rank | 전체 이벤트 내 상대 순위 |

---

### 2. 기간별 성과 비교

최근 3개월, 최근 1개월, 주차별 데이터를 함께 분석하여  
단순 현재 성과가 아니라 장기 대비 단기 변화와 최근 흐름을 함께 판단합니다.

    B --> E["이벤트별 KPI 비교"]
    C --> E
    D --> E

    E --> F["성과 변화 판단"]
